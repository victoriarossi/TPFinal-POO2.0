**README**

Pasos a seguir para la una buena implementación:

1. Tener un programa que corra JAVAFX (como Intellij)
2. Para ejecutar el programa:
    a. Tener instalado una version de Java mayor o igual a la 8. Si se tiene una version distinta a la 8, debe seguir los pasos indicados
       en la siguiente pagina: https://www.jetbrains.com/help/idea/javafx.html
    b. Se corre la clase AppLauncher.
3. Se abre el programa y allí se van a encontrar los botones correspondientes.
    a. Para realizar una figura que no sea la linea, se debe hacer un click en algun lugar del canvas y arrastrarlo hacia abajo y a la derecha.
    b. Una linea puede realizarse en cualquier direccion. Es decir se hace un click en el canvas y se arrastra para donde se quiera el mouse.
    c. En caso de que las figuras no se realicen de forma esperada se lanzará una advertencia indicandolo.
    d. Para seleccionar una o mas figuras se debe hacer un click por fuera de la misma y arrastrar en forma diagonal el mouse de manera tal que se
     forme un rectangulo imaginario el cuale engloba TODA/S la/s figura/s. Si se quiere una seleccionar una unica figura ademas de hacer un rectangulo
     se puede hacer click sobre la misma.
       d.1. Al seleccionar una o mas figuras el borde de las mismas se verá rojo.
    e. Para deseleccionar la seleccion previa basta con soltar el mouse por fuera de la seleccion o clickear algun boton que genere una nueva figura
     o elimine lo seleccionado.
       e.1. En el caso particular de la figura "Linea" para deseleccionarla y poder elegir una nueva figura es necesario volver a clickear
           el boton "Seleccion".
    f. Para asignarle un color y un grosor querido, debe primero seleccionar la figura/las figuras y luego indicar los valores queridos.
       El color del borde se vera luego de soltar la seleccion.
    g. Para cambiar el ordenamiento de las figuras (al frente o atras) primero se debe seleccionar las figuras a modificar y luego apretar el
    boton correspondiente.
4. Esperamos que el programa sea de su agrado.
